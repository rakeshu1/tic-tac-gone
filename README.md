# 🌀 Tic-Tac-Gone

A futuristic twist on tic-tac-toe — **your 4th move erases your 1st!**

🎮 **[Play it live](https://rakeshu1.github.io/tic-tac-gone/)**

## ✨ Features

- 🎯 2-Player local mode
- 🤖 vs Computer with 3 difficulty levels
- 👁️ Optional vanish hint badges to plan ahead
- 🏆 Score tracking + match history
- 🎨 3 stunning themes (Dark Neon / Pastel Light / Cyberpunk)
- 🔊 Sound effects
- 📱 Installable PWA — works offline
- 💎 Glassmorphism UI with neon glows and animations

## 🎮 How to play

Each player can have only **3 marks** on the board at any time.
When you place your 4th mark, your **oldest mark vanishes**!

Plan your moves carefully — you can win, but the board keeps shifting.

## 🛠️ Tech

- Vanilla HTML, CSS, JavaScript (no frameworks)
- Single-file architecture
- Web Audio API for sounds
- Service Worker for offline play
- LocalStorage for persistence
